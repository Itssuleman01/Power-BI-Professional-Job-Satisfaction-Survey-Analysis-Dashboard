Project Overview

This project presents an interactive Power BI dashboard designed to analyze professional job satisfaction using survey-based data. The dashboard provides insights into factors that influence employee satisfaction, including salary, work-life balance, career growth, job roles, and industry trends.

The goal of this project is to transform raw survey data into meaningful visualizations that help organizations and stakeholders better understand workforce satisfaction and identify opportunities for improvement.




---

 Tools & Technologies

* **Power BI**
* **Power Query**
* **DAX (Data Analysis Expressions)**
* **Microsoft Excel / CSV Dataset**

---

 Dashboard Features

 Job Satisfaction Overview

* Overall satisfaction ratings
* Satisfaction distribution across respondents

 Salary Analysis

* Salary comparison by job role
* Salary vs satisfaction relationship
* Income distribution analysis

 Work-Life Balance Insights

* Work-life balance ratings
* Impact of work-life balance on satisfaction


 Interactive Visualizations

* Dynamic filters and slicers
* Drill-down analysis
* KPI cards and summary metrics
* Responsive dashboard navigation
